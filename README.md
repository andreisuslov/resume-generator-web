# Resume Generator

A web-based resume generator with a form wizard, YAML editor, AI-powered tailoring, and PDF export.

**Live app: https://andreisuslov.github.io/resume-generator-web/**

## Features

- **Step-by-step wizard** - Guided form to build your resume from scratch
- **YAML editor** - Direct editing of resume data in YAML format
- **AI tailoring** - Paste a job description and let AI optimize your resume with a side-by-side diff view
- **PDF export** - Download or print your resume as a properly formatted PDF
- **Upload/download** - Import and export resume YAML files
